Answerbook App

Duration

January 2021 - May 2021

Client

Answerbook (An educational startup)

Platform

Web / Mobile

Technologies

NextJS, NodeJS, ParseServer, PostgreSQL, React Native

Code Symphony helped Answerbook (South Korean educational startup) in building this platform which can be used by students to learn something out of the syllabus. The app has two main functions:

1) An assignment grading system for teachers and students. (Admin App)

2) A social media platform for students to upload and share their textbook solutions with the community. (Web App)

Answerbook Web App

This app was mainly built for the students who can perform various activities as below:
Add Textbook & solution for the textbook
Upload solutions with own explanation
View aggregated statistics about own performance
Add & view comments & likes to the solution
View trending solution from the community, follow other students from the community

Answerbook Admin App

This app was mainly built for the teachers & admins who can perform various activities as below:

Add Textbook & solution for the textbook
View question wise statistics about student's performance
Manage student's information

Development Work

We have used efficient set of tools for web app. In this project we need to reload the data in some times like in hours, in week, in day for that we have implemented cloud functions in parse which will reload the data in given time period. Next js was used for main code base for front-end and node js for back-end. We have used HTML and CSS to make web app attractive.

The core features of the web app included an onboarding process for students, the ability to add textbook and its solution, view aggregated statistics about student's performance, add and view likes to the textbook solution, view trending solution from the community and follow other students from the community.
